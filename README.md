# Chelsea Nguyen — Personal Portfolio

Welcome to my personal portfolio! This project showcases my journey as a Computer Science 
student at UMass Boston and highlights the projects I've worked on, the skills I've developed, 
and the experiences that have shaped me as a developer.

🌸 Live site: [chelsea.cchelseanguyen.workers.dev](https://chelsea.cchelseanguyen.workers.dev)

> Built on top of [cupidbity/portfolio-template](https://github.com/cupidbity/portfolio-template) — thank you for the beautiful base! 🩷

---

## About the Portfolio

This portfolio is a reflection of my passion for web development, software engineering, and 
UI/UX design. It blends creativity with technical skills through a scrapbook-inspired aesthetic 
— complete with sticky notes, stickers, a friends & family panoramic, and a journal spread. 
It also features full Vietnamese language support, a nod to my Viet-American identity.

---

## Features

- **Responsive Design** — Fully responsive across mobile and desktop
- **Dark / Light Mode** — With system preference detection
- **Language Toggle** — English & Vietnamese (EN / VI)
- **Scrapbook Aesthetic** — Custom sticky notes, stickers, and hand-picked assets
- **Animated Stickers** — Scroll-driven sticker animations
- **Friends & Family Section** — Horizontal panoramic photo collage
- **Journal Spread** — Interactive photo carousel
- **Project Pages** — Detailed pages with screenshot carousels and tech stacks
- **Smooth Animations** — Powered by GSAP and React Spring

---

## Technologies Used

- **React 19 & TypeScript** — Component structure and type safety
- **Vite** — Fast development and build tooling
- **Tailwind CSS** — Utility-first styling
- **GSAP & React Spring** — Animations and transitions
- **React Router DOM** — Client-side routing
- **Cloudflare Workers** — Deployment and hosting
